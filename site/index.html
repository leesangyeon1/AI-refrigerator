🧊 AI Refrigerator

A CC Switch–style preset manager for AI coding agents. Collect skills, plugins, MCP servers, agents and CLAUDE.md templates into a "refrigerator," then combine them into purpose-built presets you save, apply and export.

★ View on GitHub
git clone https://github.com/leesangyeon1/AI-refrigerator.git && cd AI-refrigerator && node server.js

Local-first: the interactive app (apply presets, detect running sessions, global plugin switching) runs on your own machine — this page is a read-only catalog preview.

🧊 The Refrigerator

Built-in ingredients across 7 types — skills, plugins, MCP servers, agents, CLAUDE.md templates, tools and CLIs.
